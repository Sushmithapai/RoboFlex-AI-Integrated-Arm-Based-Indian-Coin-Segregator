# RoboFlex-AI-Integrated-Arm-Based-Indian-Coin-Segregator
This repository contains the complete source code, hardware control logic, and project files for an automated coin-sorting system that uses AI (YOLO + OpenCV) and an ESP32-controlled robotic arm to identify, pick, and place Indian coins (₹1, ₹2, ₹5, ₹10) into their respective bins.
